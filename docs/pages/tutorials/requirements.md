## Requirements

### Entity Framework Version

- Entity Framework Core 2.x
- Entity Framework 6.x
- Entity Framework 5

### Database Provider

- SQL Server 2008+
- SQL Azure
- SQL Compact
- Oracle
- MySQL
- PostgreSQL
- SQLite

### Note

- Entity Framework Core doesn't support all provider yet
- EF4 is under development
