## Requirements

### Entity Framework Version

- Entity Framework 6.x
- Entity Framework 5

### Database Provider

- SQL Server 2008+
- SQL Azure
- SQL Compact
- Oracle
- Oracle DevArt
- MySQL
- PostgreSQL
- SQLite

### Note

- Entity Framework Core is under development _(We hope to release a first version in 2017 Q2)_
- EF4 is under development
